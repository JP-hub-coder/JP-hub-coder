## 🧠 Mi camino como dev

```text
Estudiar → Construir → Debuggear → Repetir
```

- **Estudiar:** conceptos, lógica y fundamentos antes de saltar al código.
- **Construir:** proyectos reales, por pequeños que sean, para aplicar lo aprendido.
- **Debuggear:** enfrentar los errores con paciencia y curiosidad.
- **Repetir:** iterar, mejorar y nunca dejar de aprender.
